# Animated Login Form
A simple animated login form created using HTML and CSS. It has a glowing animated background and a clean login interface.

## Features
- Animated background
- Responsive layout
- Floating input labels
- Simple and clean design
- Built with HTML and CSS only

## Files
```
├── index.html
└── style.css
```

## How to Use
1. Download or clone the repository.
2. Open `index.html` in your browser.
